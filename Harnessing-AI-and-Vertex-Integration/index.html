Harnessing AI and Vertex Integration

Author: Kesari Kumar Durbaka
Download PDF Back to all papers

Abstract

This paper describes an end-to-end design for integrating Vertex tax determination with Oracle E-Business Suite, augmented by AI to reduce manual exceptions and speed close. It outlines the flow from transaction creation through tax request and response, error handling, and posting in AR, AP, and OM. The design addresses jurisdiction resolution, code mapping, rounding rules, and performance. The AI layer supports rule tuning, exception routing, and data quality hints while keeping a strong control framework for audit. The paper includes deployment options, a verification checklist, and operational metrics so teams can track improvements in exception rates, rework, and cycle time.
